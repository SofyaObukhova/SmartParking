# SmartParking - Система управления паркингом

Итоговый проект по дисциплинам "Базы данных" и "Системный подход в разработке ПО".

## Инструкция по развертыванию
1. **База данных:**
   - Для MS SQL Server: Выполнить скрипт `MSSQL_ParkingDB.sql`.
   - Для PostgreSQL: Выполнить скрипт `Postgres_ParkingDB.sql`.
2. **Настройка подключения:**
   - Откройте файл `config.properties` в корне проекта.
   - Укажите тип БД в параметре `current.db` (mssql или pg).
   - Введите ваш логин и пароль от локальной или облачной СУБД.
3. **Запуск:**
   - Откройте проект в NetBeans IDE.
   - Добавьте драйверы из папки `Libraries` в свойства проекта.
   - Запустите файл `MainFrame.java`.

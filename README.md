# SmartParking v7.7 PRO
Итоговый проект по курсу "Базы данных" и "Системный подход в разработке ПО".

## Описание
Автоматизированная информационная система для администратора платной парковки. 
Поддерживает работу с двумя СУБД (MS SQL Server и PostgreSQL), расчет стоимости, 
гибкую систему тарифов и ведение истории посещений.

## Инструкция по развертыванию (для тестировщика)

### 1. Подготовка Базы Данных
* **Если используете MS SQL Server:** Выполните скрипт `MSSQL_ParkingDB.sql` в среде SSMS. Он создаст базу `ParkingDB` и заполнит её тестовыми данными.
* **Если используете PostgreSQL:** Выполните скрипт `Postgres_ParkingDB.sql` в DBeaver (схема `obuhova_ss`).

### 2. Настройка приложения
* Откройте проект в **NetBeans IDE**.
* Перейдите в файл `config.properties`.
* В строке `current.db` укажите нужную СУБД (`mssql` или `pg`).
* Проверьте настройки подключения (хост, логин и пароль к вашей БД).

### 3. Подключение библиотек
* Убедитесь, что в папке `Libraries` проекта подключены драйверы:
  - `mssql-jdbc-9.4.1.jre8.jar` (для SQL Server)
  - `postgresql-42.7.2.jar` (для Postgres)

### 4. Запуск
* Запустите файл `src/ru/parking/MainFrame.java` (Run File).
